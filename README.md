# SCAMGUARD
AI 詐騙郵件偵測系統 v2.0 — 本地測試版

下載後的使用步驟：
第一步：取得 API Key
前往 console.anthropic.com → API Keys → 建立一組 sk-ant-... 的 Key
第二步：開啟檔案
直接用瀏覽器（Chrome / Edge / Firefox）開啟 scamguard.html
第三步：輸入 API Key
貼入頁面上方的 Key 欄位，點擊「儲存」
第四步：開始偵測
可用四個範例按鈕快速測試，或貼入您自己的可疑郵件內容

注意： API Key 只存在瀏覽器記憶體，關閉頁面即清除，不會外洩。每次偵測會消耗少量 API 費用（約 $0.001 美元/次）。
